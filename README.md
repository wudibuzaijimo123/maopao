# 冒泡排序 (Bubble Sort)

用 C 语言实现的冒泡排序算法。

## 算法原理

通过重复遍历数组，依次比较相邻元素，如果顺序错误就交换它们。每一轮遍历都会将当前未排序部分的最大值"冒泡"到末尾。

## 使用方法

```bash
gcc bubble_sort.c -o bubble_sort
./bubble_sort
```

## 运行结果

```
排序前: 64 34 25 12 22 11 90
排序后: 11 12 22 25 34 64 90
```

## 算法复杂度

| 复杂度 | 值 |
|--------|------|
| 时间复杂度（最坏） | O(n²) |
| 时间复杂度（最好） | O(n) |
| 空间复杂度 | O(1) |
